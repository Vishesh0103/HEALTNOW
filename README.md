# HealthNow – AI Health Chatbot

## What it is
HealthNow is a simple and interactive health chatbot built using Google Gemini 3.5 Flash and Streamlit.
It is designed to provide quick, reliable, and user-friendly health guidance through conversational AI.

## Tech Stack
- Streamlit
- Gemini API

## Features
- Chat-based interaction
- Basic health advice
- Safety disclaimer
